# W Fight!

## Description

_W Fight!_ is a Shooter PvP Minigame developed by _Effective_.

## Key Features

 - 2 Players
 - PvP
 - Shooter
 
## Controls

**Player 1**
-  WASD - Movement
- F - Teleport
- G - Shoot
 
**Player 2**
 - _Arrows_ - MOVEMENT
 - K - Teleport
 - L - Shoot
